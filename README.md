The goal of this repository was to:

-Understand Git fundamentals beyond just basic commands

-Practice working with local and remote repositories

-Build confidence using Git through both CLI and GUI tools

-Simulate real-world workflows involving branching, merging, and syncing

What I Practiced---

--Git (CLI – Terminal / WSL)--

Repository initialization

Staging & commit workflow

Viewing commit history (git log)

Branch creation and switching

Merging branches

Handling merge conflicts

Using .gitignore

Revert vs Reset

Rebasing (introductory practice)


--GitHub (Remote Workflow)--

Connecting local repository to remote

Push & pull system

Syncing branches

Managing clean commit history

Understanding contribution workflows


--GUI Alternative – SourceTree--

Visualizing branch history

Managing commits

Performing merges

Reverting changes

Tracking remote branches
